Приложение Grill&Bar используется для бронирования столов в одноименном ресторане Grill&Bar.
При входе в приложение пользователю предлагается забронировать стол либо по звонку в ресторан, либо через приложение.
 
При использовании услуги через приложение клиент может посмотереть описание и фотографии столов, выбрать подходящий для него стол и для продолжения бронирования 
выбрать дату и время посещения, а также указать период действия брони. После этого бронь будет создана.

Приожение запускается открытием через браузер файла index.html. Используется адаптивная верстка.
